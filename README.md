# Tree Insurance Project

## الفصل بين الواجهة الأمامية والباك إند

- `backend/` يحتوي على خدمة Node.js التي تتصل بتليجرام.
- ملفات الموقع الرئيسية (HTML, CSS, JS) تبقى في جذر المشروع كواجهة ثابتة.

## كيفية التشغيل

1. انتقل إلى مجلد الباك إند:

```bash
cd backend
```

2. ثبّت الاعتمادات:

```bash
npm install
```

3. أنشئ ملف `.env` بناءً على `.env.example` واملأ بيانات `TELEGRAM_BOT_TOKEN` و `TELEGRAM_CHAT_ID`.

4. شغّل السيرفر:

```bash
npm start
```

5. استخدم واجهة الويب من ملفات الـ HTML في الجذر مع نقطة النهاية `POST /api/send-message`.

## ملاحظة

لقد تم تحويل جميع طلبات Telegram في الواجهة الأمامية إلى الاتصال بالباك إند الجديد.
المفاتيح السرية لم تعد موجودة في ملفات HTML أو JS الخاصة بالواجهة الأمامية.
